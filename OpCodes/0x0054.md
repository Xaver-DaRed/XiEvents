# OpCode: 0x0054

| Information               | Notes |
|---                        |---    |
| **OpCode**                | `54 ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ??` |
| **OpCode Size**           | `13`  |
| **Sets `RetFlag`?**       | `Yes` |

## Description

Waits for the zone schedular to finish its current action.

    }
