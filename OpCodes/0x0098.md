# OpCode: 0x0098

| Information               | Notes |
|---                        |---    |
| **OpCode**                | `98`  |
| **OpCode Size**           | `1`   |
| **Sets `RetFlag`?**       | `Yes` |
| **Sets `ExecPointer`?**   | `Yes` |

## Description

Yields if the zone is loading data, continues otherwise.

## Pseudo Code

```cpp
void __thiscall FUNC_XiEvent_OpCode_0x0098(xievent_t* this)
{
    if (FUNC_XiZone_IsReadingExtData(PTR_Zone))
        this->RetFlag = 1;
    else
        this->ExecPointer++;
}
```
